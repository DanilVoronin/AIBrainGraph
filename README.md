# 🧠 AI Brain Blocks Constructor

Визуальный конструктор логики NPC для Unity с графовым редактором состояний, настраиваемыми действиями и условными переходами.

## 📖 О проекте

**AI Brain Blocks Constructor** — это инструмент для проектирования поведения NPC в Unity без хардкода графов состояний. С помощью визуального редактора вы создаёте взаимосвязи между состояниями, привязываете к ним последовательности действий, настраиваете условия переходов и экспортируете результат в Unity-совместимый формат.

Архитектура спроектирована для **лёгкого расширения**: новые действия, условия и типы нод регистрируются через простой C#-интерфейс или атрибуты, что позволяет адаптировать систему под любую игровую механику (стелс, бой, диалоги, патрулирование, реакция на окружение).

## ✨ Ключевые возможности

- 🎨 **Визуальный редактор нод** — проектирование графов состояний NPC в удобном drag-and-drop интерфейсе
- ⚡ **Последовательности действий** — привязка к состоянию цепочки игровых логики: движение, анимации, атаки, поиск цели, кастомные C#-скрипты
- 🔀 **Условные переходы** — настройка ветвления на основе параметров игрового мира (`Health`, `DistanceToTarget`, `LineOfSight`, кастомные предикаты)
- 🔌 **Расширяемость через C#** — регистрация новых `Action` и `Decision`

## 🚧 Текущий статус

Проект находится на стадии **Alpha / черновой реализации**. Базовая структура редактора, ядро обработки графов и интерфейс для кастомных действий/условий уже работают.
